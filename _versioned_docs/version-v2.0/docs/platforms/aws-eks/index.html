AWS EKS

Ondat is fully compatible with AWS EKS. To install Ondat on EKS, please follow our Kubernetes installation instructions page.

An EKS deployment of Kubernetes uses AWS Linux by default with an optimized kernel. As the requisite kernel modules are not available for Ondat to use TCMU, FUSE will be used as a fallback. Using FUSE instead of TCMU has performance implications.

For more details about the OS Distributions check the System Configuration page.
